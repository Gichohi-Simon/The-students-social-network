#The Students Social Network

#installation

```
    yarn install
```

running the project
```
npm run start 

for both client and server
```

#Functionality

user can: 
    signUp, using googleOauth and the conventinal way 
    create a post
    like their own post and other peoples post
    edit their own post

ps
Project uses Mvc pattern in Backend
